# 🌿 Landing Page — Profissionais de Saúde

Projeto de landing page desenvolvido como portfólio, voltado para profissionais da área da saúde (farmacêuticos, médicos, nutricionistas, psicólogos e outros).

---

## 📋 Sobre o Projeto

Uma landing page moderna, responsiva e elegante criada especialmente para profissionais de saúde que desejam ter uma presença digital profissional e atrair mais pacientes online.

O projeto foi desenvolvido com foco em:
- Design limpo e transmissor de confiança
- Seções estratégicas para conversão de visitantes em pacientes
- Totalmente responsivo para mobile e desktop
- Animações suaves para melhor experiência do usuário

---

## ✨ Funcionalidades

- **Navegação fixa** com link direto para agendamento
- **Seção Hero** com cartão de apresentação profissional
- **Barra de credibilidade** com formação, registro e disponibilidade online
- **Grade de serviços** com 6 cards animados
- **Seção Sobre** com diferenciais e formação
- **Depoimentos** de pacientes
- **Formulário de agendamento** com CTA estratégico
- **Scroll reveal** — animações ativadas pela rolagem da página

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, grid, flexbox, animações e responsividade
- **JavaScript** — scroll reveal com IntersectionObserver
- **Google Fonts** — Cormorant Garamond + DM Sans

## 🔧 Personalização

Para adaptar a landing page para outro profissional de saúde, edite no `index.html`:

- **Nome e CRM/CRF** — seção Hero e rodapé
- **Especialidade** — tag de destaque no Hero
- **Estatísticas** — anos de experiência, pacientes atendidos
- **Serviços** — cards na seção de serviços
- **Depoimentos** — nomes, textos e cidades
- **Cores** — variáveis CSS no `:root`
- **E-mail de contato** — rodapé
